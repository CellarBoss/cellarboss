# Winemakers

Winemakers represent wine producers or estates. Each [wine](/resources/wines) is associated with a winemaker.

## List View

Access Winemakers from **More > Reference Data > Winemakers**.

![Winemakers list](/screenshots/winemakers-list.png)

### Search

Use the search bar to filter winemakers by name.

## Creating a Winemaker

Tap the **+** button to add a new winemaker.

| Field | Required | Description          |
| ----- | -------- | -------------------- |
| Name  | Yes      | The winemaker's name |

## Viewing and Editing

Tap a winemaker to view details, then tap **Edit** to modify.

## Deleting a Winemaker

Tap **Delete** on the detail screen. You will be asked to confirm.

::: warning
Deleting a winemaker will also remove all associated wines, vintages, and bottles.
:::
