# Grapes

Grapes represent grape varieties. Wines can be associated with one or more grape varieties.

## List View

Access Grapes from **More > Reference Data > Grapes**.

![Grapes list](/screenshots/grapes-list.png)

## Creating a Grape

Tap the **+** button to add a new grape variety.

| Field | Required | Description              |
| ----- | -------- | ------------------------ |
| Name  | Yes      | The grape variety's name |

## Viewing and Editing

Tap a grape to view details, then tap **Edit** to modify.

## Deleting a Grape

Tap **Delete** on the detail screen. You will be asked to confirm.
