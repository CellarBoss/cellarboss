# Countries

Countries are used to organise [regions](/resources/regions) geographically.

## List View

Access Countries from **More > Reference Data > Countries**.

![Countries list](/screenshots/countries-list.png)

## Creating a Country

Tap the **+** button to add a new country.

| Field | Required | Description        |
| ----- | -------- | ------------------ |
| Name  | Yes      | The country's name |

## Viewing and Editing

Tap a country to view details, then tap **Edit** to modify.

## Deleting a Country

Tap **Delete** on the detail screen. You will be asked to confirm.

::: warning
Deleting a country will also remove all associated regions, wines, vintages, and bottles.
:::
