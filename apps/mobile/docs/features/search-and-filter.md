# Search, Sort & Filter

Most list screens in CellarBoss Mobile include search, sort, and filter controls to help you find records quickly.

## Search

The search bar at the top of each list filters results as you type. Search matches against the primary field (e.g. wine name on the Wines screen, bottle wine name on the Cellar screen).

## Sort

Tap the sort icon to open the sort sheet. Select a sort option to reorder the list. Available sort options vary by screen.

## Filter

Tap the filter icon to open the filter sheet. Select one or more values across filter categories to narrow results. Active filters are shown as a count badge on the filter icon.

Available filters vary by screen:

| Screen   | Available Filters                            |
| -------- | -------------------------------------------- |
| Cellar   | Status, Wine Type, Drinking Window, Location |
| Wines    | Type, Winemaker, Region                      |
| Storages | Location                                     |

## Pull to Refresh

Pull down on any list to refresh the data from your server.
