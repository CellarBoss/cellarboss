# Locations

Locations represent physical places where wine is stored — such as a home cellar, off-site storage, or a restaurant. Each location can contain multiple [storages](/resources/storages).

## List View

![Locations list](/screenshots/locations-list.png)

### Columns

| Column | Description       |
| ------ | ----------------- |
| Name   | The location name |

## Creating a Location

Click **Create** to add a new location.

![Create location](/screenshots/locations-create.png)

### Fields

| Field | Required | Description                                                 |
| ----- | -------- | ----------------------------------------------------------- |
| Name  | Yes      | The location name (e.g., "Home Cellar", "Off-site Storage") |

## Viewing a Location

Click any row to view the location's details and its associated storages.

![Location detail](/screenshots/locations-detail.png)

## Editing a Location

From the detail view, click **Edit** to modify the location name.

## Deleting a Location

::: warning
A location cannot be deleted if it has associated storages. Remove the storages first.
:::
