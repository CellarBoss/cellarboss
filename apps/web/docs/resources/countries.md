# Countries

Countries represent wine-producing nations. Each country can contain multiple [regions](/resources/regions).

## List View

![Countries list](/screenshots/countries-list.png)

### Columns

| Column  | Description                                                                           |
| ------- | ------------------------------------------------------------------------------------- |
| Name    | The country name                                                                      |
| Actions | Shortcut buttons for common actions. Hover over the icon to see what each button does |

## Creating a Country

Click **Create new Country** to add a new country.

![Create country](/screenshots/countries-create.png)

::: info
Country names must be unique
:::

### Fields

| Field | Required | Description      |
| ----- | -------- | ---------------- |
| Name  | Yes      | The country name |

## Viewing a Country

Click the name of a country to view the country's details.

![Country detail](/screenshots/countries-detail.png)

## Editing a Country

Click the **Edit** button to modify the country name.

## Deleting a Country

::: warning
A country cannot be deleted if it has associated regions. Remove or reassign the regions first.
:::
